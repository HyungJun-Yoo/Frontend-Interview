## this

### this의 정의

- 함수 내부에서 사용되며, 함수를 호출하는 방법에 따라 결정되는 참조 대상 객체입니다.

### 바인딩이란?

- 바인딩(binding)은 변수, 함수 등의 이름과 값을 연결하는 과정을 의미한다.
- this 바인딩은 this가 참조하는 객체를 결정하는 과정을 의미한다.

### this 바인딩 우선순위

1. new 바인딩

2. 명시적 바인딩  ```call, apply, bind```

3. 암시적 바인딩

4. 전역 객체 바인딩
